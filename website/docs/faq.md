# FAQ

Frequently asked questions about CodeVerify.

## General

### Does CodeVerify store my code?

No. Code is processed in memory and never persisted. Only metadata about findings is stored.

### What languages are supported?

Currently Python and TypeScript. Go is in beta. Java and Rust are on the roadmap.

### How long does analysis take?

Most files complete in 2-5 seconds. Large PRs (500+ lines) may take up to a minute.

## Pricing

### Is CodeVerify free?

CodeVerify is open source (MIT license). Self-hosting is always free. A managed cloud service with additional features is available.

### What's the cost of the cloud service?

See our [pricing page](https://codeverify.dev/pricing) for current plans.
